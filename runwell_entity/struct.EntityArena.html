pub struct EntityArena<T> { /* fields omitted */ }

Primary map to create new entities and store required data for them.

Note

Use this if all your entities are required to store some additional necessary information or data with them or if the data identifies the entity.

For efficiency and safety reasons it is not possible to remove entities.
