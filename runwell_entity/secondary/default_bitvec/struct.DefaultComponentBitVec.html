pub struct DefaultComponentBitVec<K> { /* fields omitted */ }

Container where every entity has a bool component that is default initialized to false.

Note

  • Use this if you need to store bool types as components for your entities efficient and expect most of the entities to have this component.
  • Prefer using this data structure if the bool components are likely false (default) for most entities.
  • The API of the DefaultComponentBitVec differs slightly from the DefaultComponentVec because it is not very useful to have bool references which hinders implementing the common Index and IndexMut traits. Use the get and set methods of the DefaultComponentBitVec type instead respectively.
  • Unbounded iteration over a default component data structure might not be what a user normally wants since they eagerly iterate over the entire key space of their entity which yields approx 2^32 components in total.
